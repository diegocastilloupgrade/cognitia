# UI spec

## Backoffice
- Cabecera, menú lateral y contenido principal.
- Listados con filtros.
- Formularios con validación.

## Ejecución
- Zona dominante para avatar y estímulo.
- Vista simple, sin ruido visual.
- Información mínima para el clínico supervisor.

## Resultados
- Vista por bloques de ítems.
- Métricas agregadas claras.
- Acceso a secuencias reconocidas cuando aplique.
