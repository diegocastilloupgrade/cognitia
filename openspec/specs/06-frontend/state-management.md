# Gestión de estado

## Estado recomendado
- Estado local para formularios sencillos.
- Servicios Angular con RxJS para sesiones activas y ejecución.
- Store ligera o NgRx solo si el equipo lo necesita por complejidad.

## Estado crítico
- Usuario autenticado.
- Paciente seleccionado.
- Sesión activa.
- Ítem en curso.
- Resultados consolidados.
