# Índice de especificaciones

## Finalidad
Este paquete define la especificación SDD del prototipo COGNITIA para Angular, Node.js y PostgreSQL, con integración de avatar Unith en modo plugin.

- Contexto y alcance.
- Requisitos funcionales y no funcionales.
- Historias de usuario.
- Dominio, estados y reglas.
- Especificación detallada de ítems 3.1 a 3.7.
- Contratos API y OpenAPI inicial.
- Arquitectura frontend y backend.
- Modelo de datos y SQL base.
- Integración con Unith.
- Estrategia de calidad, despliegue y entrega.
