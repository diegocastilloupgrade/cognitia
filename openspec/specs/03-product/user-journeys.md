# User journeys

## Journey clínico
1. Inicia sesión.
2. Accede a pacientes.
3. Crea o selecciona paciente.
4. Crea sesión.
5. Lanza la ejecución.
6. Supervisa la prueba.
7. Consulta resultados.

## Journey paciente
1. Entra en la pantalla de sesión preparada.
2. Escucha instrucciones del avatar.
3. Responde a los ítems.
4. Finaliza la sesión.
