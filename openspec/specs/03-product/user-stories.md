# Historias de usuario

## Autenticación
- Como usuario interno, quiero iniciar sesión para acceder al sistema.
- Como usuario interno, quiero recuperar mi contraseña si la he olvidado.

## Pacientes
- Como clínico, quiero dar de alta un paciente para poder crear sesiones.
- Como clínico, quiero editar un paciente para mantener sus datos actualizados.
- Como clínico, quiero borrar un paciente no utilizado para mantener el listado limpio.

## Sesiones
- Como clínico, quiero crear una sesión en borrador para preparar una prueba.
- Como clínico, quiero impedir sesiones abiertas duplicadas por paciente.
- Como clínico, quiero iniciar una sesión preparada para comenzar la prueba.

## Ejecución
- Como paciente, quiero recibir instrucciones claras del avatar para realizar el test.
- Como sistema, quiero registrar tiempos, respuestas y resultados por ítem.
- Como sistema, quiero finalizar automáticamente los ítems por tiempo cuando corresponda.

## Resultados
- Como clínico, quiero revisar resultados por ítem para valorar la sesión.
