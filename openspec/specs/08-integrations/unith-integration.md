# Integración Unith

## Objetivo
Usar Unith para representar el avatar, emitir instrucciones y obtener texto reconocido por ASR, mientras el backend propio controla la lógica funcional.

## Modo recomendado
Plugin mode.

## Necesidades
- Gestión de token.
- Arranque de sesión del avatar.
- Envío de mensajes e instrucciones.
- Recepción de eventos y texto ASR.
