# Ítem 3.3 Control mental auditivo

## Objetivo
Evaluar atención, flexibilidad mental y memoria operativa mediante alternancia número-letra.

## Precondiciones
- Superar recordatorio de abecedario A-L.
- Superar conteo 1-12.

## Reglas
- Si no supera prepruebas, el ítem queda como NO_HA_PODIDO_COMPLETARLO.
- Se registra la secuencia completa producida por el paciente.
- Se contempla la variante CH según el cuestionario.
- Si empieza por letra, el avatar corrige la consigna al llegar al 2.

## Datos mínimos
- completion_state
- total_seconds
- hits
- errors
- recognized_sequence
