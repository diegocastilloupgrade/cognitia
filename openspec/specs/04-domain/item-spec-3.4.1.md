# Ítem 3.4.1 Interferencia de números

## Objetivo
Evaluar interferencia cognitiva en dos subpruebas sobre cartas numéricas.

## Reglas
- Cada subprueba tiene 3 ejemplos previos.
- Cada subprueba dura 45 segundos.
- El avatar no corta al paciente mientras habla.
- Se registra el número de ítems realmente presentados.

## Datos mínimos
- items_presented
- hits
- errors
