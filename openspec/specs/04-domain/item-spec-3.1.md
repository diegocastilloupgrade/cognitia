# Ítem 3.1 Memoria visual

## Objetivo
Registrar la denominación correcta de cuatro imágenes sin anticipar que deberán recordarse después.

## Estímulos
- Guitarra.
- Zapato.
- Moto.
- Jirafa.

## Reglas
- Se muestra cada imagen sobre fondo blanco.
- En silencio, el avatar puede reforzar con la misma pregunta.
- Si el paciente falla, el avatar confirma el nombre correcto.
- Nunca se menciona que deba recordarlas.

## Datos mínimos
- hits
- errors
- recognized_answers_by_image
