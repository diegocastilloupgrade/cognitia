# Ítem 3.6 Recuerdo libre

## Objetivo
Medir recuerdo espontáneo de las imágenes iniciales.

## Reglas
- No importa el orden de mención.
- Si el paciente dice que se ha quedado en blanco, el avatar le deja unos segundos.
- Si acierta todo, el avatar felicita.

## Datos mínimos
- total_seconds
- hits
- errors
- recognized_answer
