# Arquitectura lógica

## Capas
- Presentación: Angular.
- Aplicación: casos de uso y workflows en Node.
- Dominio: reglas del test, estados y scoring.
- Infraestructura: PostgreSQL, correo, integración con Unith.

## Principio clave
La lógica de ejecución del test no depende de comportamiento implícito del avatar; debe residir en el backend propio.
